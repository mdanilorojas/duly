Detalles de implementación

Fuera de alcance
  • Upload, hashing, custodia, SSE, folds de eventos (diffRuns) — pack-kit / consola.
  • Vocabulario ISO hardcodeado en el DS — entra por props/stories.
  • La consola como app — este repo entrega DS + demo Storybook.
Testing (vitest + Testing Library + jest-axe, patrón del repo)
  • BandGauge: segmentos = value, aria de meter, 3 rangos de tono, axe.
  • AgentStatusMatrix: API previa intacta, compact, critical accesible, onSelectItem → botones.
  • KanbanBoard: onMove(ticketId, toColumn, index) al soltar, read-only sin onMove, announcements, axe.
  • Dropzone: onFiles por drop y por input, drag-over, disabled inerte, axe.
  • DeltaList / ConnectorStatus / Stepper: render + semántica aria + axe.
Convenciones heredadas
  • Tone de trace-log.variants — nunca color crudo. cn de @/lib/utils. useCopy para strings.
  • JSDoc español con referencia de área NORTH_STAR. Kebab-case, named exports, story co-locada.
  • NORTH_STAR.md gana filas nuevas (área C y donde corresponda).

Criterio de éxito: cada bloque del wireframe v0 tiene componente o receta asignada sin GAP restante · story "Consola ISO completa" replica las 5 pantallas + shell en tema cockpit · pnpm test y lint verdes · NORTH_STAR actualizado.
