Arreglo de fondo — contexto WebGL compartido

El navegador tiene un techo de ~16 contextos WebGL vivos. Hoy cada tarjeta crea el suyo. Con 6 por página no se nota, pero es un límite real del primitivo AgentCore. Se soluciona con un singleton a nivel de módulo (no un Context de React — cero props nuevas, cero wrapper exigido a nadie):

  • Un solo canvas WebGL "maestro", offscreen, creado la primera vez que monta un AgentCore.
  • Cada tarjeta pinta en su propio canvas 2D visible, proyectando el maestro vía drawImage cada frame.
  • Un solo requestAnimationFrame compartido recorre todas las tarjetas activas.
  • El pausado por IntersectionObserver (agregado hoy mismo en la auditoría) se conserva igual.
  • API pública de AgentCore sin cambios — ningún consumidor se entera.
