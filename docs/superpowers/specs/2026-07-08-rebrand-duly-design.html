Why Duly

From "duly noted / duly authorized / duly executed" — the ritual adverb of contracts and audit trails. The product's whole pitch is that every agent action is debidamente registered, approved, on record. Pronounces the same in English and Spanish. Checked clean against npm scope, npm package, and web search (only collision: Duly Health & Care, a Chicago healthcare system — different industry, accepted).
